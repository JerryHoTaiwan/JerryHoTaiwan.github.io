Recovering a useful PSF matters more than recovering the exact optical model.

This site presents the motivation, method, and results from the unknown-perturbation lens self-calibration study. Research question

Lens self-calibration asks whether the effective point spread function, or PSF, of an optical system can be inferred directly from a degraded image and then used for restoration. In this project the input is a single noisy blurred image y_n, produced by an unknown perturbed lens instance, and the goal is to recover both the latent clean image x_clean and the blur kernel k.

y = k * x + n

The challenge is that the true perturbation is outside the assumed CameraLensS4 parameter family. The exposed calibration model does not include surface-2 curvature, but the ground truth does. Exact physical self-calibration is therefore impossible by construction. The practical question becomes whether a blind method can still recover a PSF that is useful for image restoration.

What the study shows

  • Self-calibration is still possible under unknown perturbation.
  • Image prior value dominates exact PSF knowledge.
  • A narrow optical PSF prior can be harmful in coupled blind sampling.
  • Iterative refinement saturates at an image-model ceiling.
  • USAF is a poor calibration target under the FFHQ image prior.
  • The best motion-blur Blind-DPS result is reproducible across faces.
