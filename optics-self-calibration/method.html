Problem statement and motivation

Lens self-calibration asks whether the effective point spread function, or PSF, of an optical system can be inferred directly from an observed degraded image, without access to a calibration target or repeated measurements. Here the input is a single noisy blurred image y_n and the goal is to recover both the latent clean image x_clean and the unknown kernel k.

y = k * x + n

The problem is ill-posed because many image-kernel pairs can explain the same observation. In the Fourier domain, Y(w) = K(w) X(w) + N(w), so attenuation in K(w) at high frequencies can always be traded against a different estimate of X(w). This is the core ambiguity of single-image blind deconvolution.

The optics setting makes the problem practically important because deployed lenses differ from nominal design due to manufacturing tolerances, decenter, tilt, spacing changes, thermal drift, and perturbations not exposed by the calibration model.

Unknown perturbation outside the assumed model

The specific challenge in this study is model misspecification. The ground-truth perturbation includes:

  • Delta c2 = -0.010
  • Delta c4 = -0.002
  • tilt_x = tilt_y = 0.010 rad

But the CameraLensS4 calibration interface only exposes c4, tilt, and d_sensor. Surface-2 curvature is not recoverable, which means exact physical self-calibration is impossible by construction.

Blind-DPS formulation

The main blind method is a coupled diffusion posterior sampler with two DDPM chains:

  • an image DDPM for x
  • a kernel DDPM for k

At each step the method predicts denoised states x_hat_0 and k_hat_0 and combines prior scores with measurement consistency:

grad log p(x_t | y_n) ~= grad log p(x_t) + grad log p(y_n | x_hat_0, k_hat_0)
grad log p(k_t | y_n) ~= grad log p(k_t) + grad log p(y_n | x_hat_0, k_hat_0)

The shared residual is r = || y_n - conv(x_hat_0, k_hat_0) ||_2, and the measurement gradients are computed with respect to each chain. Both are scaled by sqrt(alpha_bar_t). A key implementation detail from the handoff is that the correction must be applied to out["sample"], not directly to x_t.

Iterative refinement formulation

The strongest hybrid result avoids unstable full coupling. Starting from an initial kernel estimate k^(0), it alternates:

  1. non-blind image DPS with fixed kernel k^(r)
  2. direct kernel fitting with fixed image x^(r+1)
k^(r+1) = argmin_k || y_n - op.forward(x^(r+1), k) ||_2

This turns the problem into alternating projection between the image manifold implied by the diffusion prior and the measurement-consistent kernel set implied by gradient-based fitting.

Experimental setup

The optical system is a Canon EF 40mm f/2.8 lens represented in DeepLens as an 11-surface model. The scene is a 256 x 256 FFHQ face with additive Gaussian noise sigma = 0.01. The degraded observation has PSNR(y_n) = 23.10 dB.

The GT PSF is a smooth diffuse blob with weighted spread 6.97 px. This is not a sparse line kernel. It is exactly the kind of blur for which classical inversion is unstable because high frequencies are strongly attenuated.

The primary metrics are:

  • PSNR(restored)
  • Delta PSNR = PSNR(restored) - PSNR(y_n)
  • Kernel MSE versus the GT PSF
